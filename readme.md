# Commands

```
start LINE - up facing buffer stop
end LINE - down facing buffer stop
join LINE - up facing entrance point
leave LINE - down facing entrance point

slip LINE_FROM LINE_TO - slip from one line to another

merge LINE_SIDE LINE_MAIN - one line joins another
split LINE NEW_LINE DIRECTION - a new line breaks off

name LINE NEW_NAME - a line changes name

setup LINE <LINE> <...> - setup initial track order
```
